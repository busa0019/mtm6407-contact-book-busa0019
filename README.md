# Contact Book Application

A Vue.js 3 contact management application with Vue Router and localStorage persistence.

## Features
- View all contacts alphabetically by last name
- Search contacts by name, email, company, or tags
- Add new contacts with detailed information
- Edit existing contacts
- Delete contacts with confirmation
- Mobile-responsive design
- All data saved to browser's localStorage

## Installation
1. Clone repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Note
Sample contacts are loaded on first visit. All user-added contacts persist in browser storage until cleared.

## Technologies
- Vue.js 3
- Vue Router 4
- Vite
- Bootstrap 5
- Bootstrap Icons
